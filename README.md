# gc-product-simplification-v2

Focused design notes for simplifying the current Gas City product surface
outside the separate noun-centric CLI redesign work.

## Contents

- `design/product-simplification.md`
  - a seed note that categorizes the current product surface and calls out
    promising simplification seams

## Intent

This repo is intentionally small. It exists to give the broader
"make the product simpler" conversation a clean home without mixing it into:

- the Pack/Registry CLI proposal
- the noun-centric CLI redesign work

The first task is not to prescribe a replacement surface. It is to build a
shared map of what GC already is, where the overlaps are, and which seams are
worth simplifying next.
